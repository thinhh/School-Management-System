# School-Management-System
## What do you need before running the app
- MYSQL database
- Java version 11
- Node (latest version)
## How to run the app
- Git clone the repository
- cd to the root folder
- Start your database by either downloading mySQL locally or on Docker Container 
- cd spring-backend -> mvn spring-boot:run 
- cd ../react-frontend -> npm start
### Instruction to start a database
- Loggin to your root user for mysql and use the password:fullstack
- Create database: fullstack
